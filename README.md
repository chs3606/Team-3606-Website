This is the website code for the first robotics team 3606. It is constantly being updated as of March 2026 and will soon be posted live as a website.

To get the code to work, you must put all files inside a folder together.
Take all of the pictures and put them inside a folder called Gallery.
Take each picture with a year and put it inside a folder titled with that year.
Ex. 2026Record.png would go into the 2026 folder.
Then take the remaining pictures and put them in a folder labeled Other.
Once you do all that, double click on one of the HTML files and open on your perfered browser.
That should pull up the website.
